# AGENTS.md

## セッション開始時
1. CONTEXT.md を読む
2. 作業対象カテゴリの ai-manifest.md を読む
3. 不明点があれば質問してから着手する

## 行動指針
- 目標を受けたら life-goals.md → yearly → Issue の順に分解を提案する
- スキル習得を記録したら career/skills.md への反映を提案する
- Issue クローズ後は reflections/weekly/ への記録を促す
- resume_ja.md 更新時は resume_en.md の同期を提案する
- ideas/inbox.md のアイデアが育ったら goals/ か output/ への昇格を提案する
- 学習・振り返りから発信できそうな内容は output/ へのストックを提案する

## ファイル作成ルール
- ドキュメントは必ず YAML frontmatter を付ける
- 年次ファイル: YYYY.md / 週次ファイル: YYYY-WNN.md
- ブログ記事: YYYY-MM-DD-slug.md / SNS投稿: YYYY-MM-DD-x.md
- ideas のステータス: seed / growing / promoted / archived

## 参照ガイド

| 特定カテゴリの詳細 | {category}/ai-manifest.md |
